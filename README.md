# nodejs-web-koa-websocket-ejs-views-attributes-hello-world

## Description
Uses MVC for web content and
updates clicks to all connected
clients in real time.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
